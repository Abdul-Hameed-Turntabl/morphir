# Get Started

1. Ensure NodeJS is installed

```sh
$ node -v
$ npm -v
```

2. Install and run Docusaurus locally

```sh
$ git clone git@github.com:finos/morphir.git
$ cd morphir/website

# Download dependencies
$ npm install

# Start
$ npm start
```

This will open a browser on http://localhost:3000/morphir/
